# ============================================================
# ProxyRules — 手动添加直连域名
# ============================================================
# 一行一个域名，将被追加到 direct_domain.txt 的输出中
# 支持以下格式:
#   example.com              — 完整域名
#   *.example.com            — 通配符（匹配所有子域名）
#   .example.com              — 域名后缀（同上效果）
#
# 使用方法:
#   1. 在此文件中添加域名，一行一个
#   2. 提交并推送，GitHub Actions 会在下次构建时自动合并
#   3. 无需修改任何脚本代码
# ============================================================
